## Alpine 镜像使用帮助

### 地址

https://mirrors.idc.daocloud.io/alpine/

### 说明

Alpine Linux 软件源

### 收录架构

x86_64

### 使用说明

一般情况下，将 `/etc/apk/repositories` 文件中 Alpine 默认的源地址 `http://dl-cdn.alpinelinux.org/` 替换为 `http://mirrors.idc.daocloud.io/` 即可。

可以使用如下命令：

#### 方法一

```shell
sed -i 's/dl-cdn.alpinelinux.org/mirrors.idc.daocloud.io/g' /etc/apk/repositories
```

#### 方法二(一键脚本)

```bash
# 使用curl
bash <(curl -sSL http://mirrors.idc.daocloud.io/mirrors.sh)
# 使用wget
bash <(wget -qO- http://mirrors.idc.daocloud.io/mirrors.sh)

## 以上使用任一方式均可
```

也可以直接编辑 `/etc/apk/repositories` 文件。以下是 v3.20 版本的参考配置：

```text
https://mirrors.idc.daocloud.io/alpine/v3.20/main
https://mirrors.idc.daocloud.io/alpine/v3.20/community
```

也可以使用 `latest-stable` 指向最新的稳定版本：

```text
https://mirrors.idc.daocloud.io/alpine/latest-stable/main
https://mirrors.idc.daocloud.io/alpine/latest-stable/community
```

### 注意

更改完 `/etc/apk/repositories` 文件后请运行 `apk update` 更新索引以生效。
