## openEuler镜像使用帮助

### 地址

[https://mirrors.idc.daocloud.io/openeuler/](https://mirrors.idc.daocloud.io/openeuler/)

### 说明

openEuler 公共镜像软件源

### 收录架构

官方支持

### 使用方法

#### 方法一

将`/etc/yum.repos.d/openEuler.repo`文件中默认的源地址 `http://repo.openeuler.org/` 替换为 `https://mirrors.idc.daocloud.io/openeuler/`

```bash
sed -i.bak 's|http://repo.openeuler.org/|https://mirrors.idc.daocloud.io/openeuler/|g' /etc/yum.repos.d/openEuler.repo
```

#### 方法二(一键脚本)

```bash
# 使用curl
bash <(curl -sSL http://mirrors.idc.daocloud.io/mirrors.sh)
# 使用wget
bash <(wget -qO- http://mirrors.idc.daocloud.io/mirrors.sh)

## 以上使用任一方式均可
```
