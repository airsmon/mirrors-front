## ArchLinuxCN 镜像使用帮助

### 地址

[https://mirrors.idc.daocloud.io/archlinuxcn/](https://mirrors.idc.daocloud.io/archlinuxcn/)

### 说明

Arch Linux CN 软件源

### 简介

> Arch Linux 中文社区仓库是由 Arch Linux 中文社区驱动的非官方软件仓库，包含许多官方仓库未提供的额外的软件包，以及已有软件的 git 版本等变种。
>
> \- [Arch Linux 中文社区仓库](https://www.archlinuxcn.org/archlinux-cn-repo-and-mirror/)

仓库地址：[http://repo.archlinuxcn.org](http://repo.archlinuxcn.org)

---

### 使用说明
在 `/etc/pacman.conf` 文件末尾添加两行：

```toml
[archlinuxcn]
Server = https://mirrors.idc.daocloud.io/archlinuxcn/$arch
```
然后请安装 `archlinuxcn-keyring` 包以导入 GPG key。

```shell
sudo pacman -Sy archlinuxcn-keyring
```

随后将自动进行 GPG 签名。
